流水线基本概念

流水线是多条指令重叠操作的准并行处理技术：

• 各部件同时处理不同指令的不同部分

• 提高部件利用率和指令平均执行速度

示例：取指(1ms)、分析(1ms)、执行(1ms)

3条指令串行执行：9ms

流水线执行：5ms

核心参数：

• 执行时间计算

• 吞吐率

• 加速比

流水线执行时间

流水线周期：执行时间最长的一段

理论公式：\[$(t_1 + t_2 + ... + t_k) + (n-1)*t$\]

实践公式：\[$k*t + (n-1)*t$\]

例题：

取指=3Δt，分析=2Δt，执行=4Δt

10条指令：

串行：(3+2+4)×10 = 90Δt

流水线：4Δt×3 + (10-1)×4 = 48Δt

流水线吞吐率

定义：单位时间完成的任务数

基本公式：

\[ TP = \frac{\text{指令条数}}{\text{流水线执行时间}} \]

最大吞吐率：

\[ TP_{\max} = \frac{1}{t} \]

例题：

取指=3Δt，分析=2Δt，执行=4Δt，10条指令吞吐率：

\[$TP = \frac{10}{48Δt} ≈ 0.208/Δt$\]

\[$TP_{max} = \frac{1}{4Δt}$\]

流水线加速比

定义：串行时间与流水线时间之比

计算公式：

\[$S = \frac{串行执行时间}{流水线执行时间}$\]

例题：

串行时间：90Δt

流水线时间：48Δt

加速比：

\[$S = \frac{90}{48} = 1.875$\]

意义：衡量流水线性能提升程度

✓ 吞吐率=指令数/流水线时间

✓ 最大吞吐率=1/周期
