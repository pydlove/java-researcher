存储器体系

主存储器

特点：

  • 与CPU直接交互
  • 存储正在运行的程序和数据
  • 断电后数据丢失

辅助存储器

特点：

  • 需要调入主存才能使用
  • 长期保存数据
  • 速度比主存慢
重要：主存速度 > 辅存速度，但容量 < 辅存
墨墨说-Java

数据流动方向

三大信息流

1. 数据流：

输入设备 → 主存 → 运算器 → 输出设备

2. 控制流：

控制器 → 所有硬件部件

3. 地址/指令流：

CPU ↔ 主存（读写指令）

核心要点总结

  1. CPU = 运算器 + 控制器
  2. 主存与CPU直接交互
  3. 数据流方向固定
  4. 控制器是系统的指挥中心
